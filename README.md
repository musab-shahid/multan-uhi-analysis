# Urban Heat Island Analysis of Multan (2013–2023)
Spatiotemporal Analysis of Urban Heat Island Effect in Multan, Pakistan (2013–2023) Using Google Earth Engine, Machine Learning, Landsat 8 Imagery, and Remote Sensing Spectral Indices

## Overview

This project investigates the Urban Heat Island (UHI) effect in Multan, Pakistan using Google Earth Engine and Landsat 8 Collection 2 Level 2 imagery.

## Tools

- Google Earth Engine
- Python
- Google Colab
- geemap
- pandas
- NumPy
- Matplotlib

## Outputs (in Colab Code pdf)

- Land Surface Temperature (LST)
- NDVI
- NDBI
- MNDWI
- K-Means Classification
- Heat Vulnerability Maps
